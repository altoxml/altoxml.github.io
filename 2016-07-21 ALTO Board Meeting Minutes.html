Minutes

No new action items.

Briefly discussed the terms of use for ALTO XML, namely, Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0). Clemens has not yet had an opportunity to talk with Armin Talke, Staatsbibliothek zu Berlin Legal Advisor for Copyright and related matters. Frederick will see Armin at WLIC 2016 and talk with him about ALTO terms of use. It was noted that the Library of Congress has not (yet?) adopted Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0).

Joachim encouraged all board members to comment on the proposed solution for the Glyphs change request (issue 26) so that the board can vote on the proposed solution and a new version of the schema incorporating the solution can be released.

No substantive discussion on the processing history change request. Issues 13, 27, 35, and 36 have been merged to create issue 39 Processing history. Clemens is issue 39's champion. He has organized a separate teleconference discussion for any board member interested in the issue. The teleconference is scheduled for UTC 14.00 July 27, 2016.

Jean Philippe, champion for issue 33 did not attend the meeting, however, Clemens mentioned that he talked with the IIIF folks. IIIF wants to adopt ALTO XML as its standard for OCR text. Accordingly, it would like the ALTO board to create / assign a format identifier for ALTO XML, for example, a MIME type like application/xml or text/xml, and also to give ALTO a label, for example, ALTO XML or ALTO OCR. Clemens added issue 40 for this change request.
